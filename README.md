# Chat Navigator - Gemini & ChatGPT

## 项目简介

Chat Navigator 是一个 Tampermonkey 用户脚本，用于在 Gemini 和 ChatGPT 对话页面中快速定位、搜索并跳转到历史对话内容。

## 功能特性

- 对话导航：自动提取当前页面中的用户消息和 AI 回复，并在浮动面板中生成导航列表。
- 点击跳转：点击列表条目后平滑滚动到对应的对话位置，并短暂高亮目标内容。
- 搜索过滤：通过搜索框按文本内容过滤对话条目，匹配时不区分大小写。
- 消息计数：面板标题旁显示当前检测到的消息数量。
- 手动刷新：通过刷新按钮重新扫描当前页面的对话内容。
- 自动监控：启用监控后，脚本会使用 `MutationObserver` 监听页面变化，并在对话内容变化时自动刷新列表。
- 初始重试刷新：启用监控后，如果首次未检测到消息，会在短时间内自动重试刷新。
- 保活注入：启用监控后，脚本会定时检查面板是否仍存在，并在需要时重新注入。
- 监控开关：监控默认关闭；关闭时不扫描 DOM、不启动自动刷新、不运行保活定时器，面板显示“监控已暂停，点击启用”。
- 状态持久化：主题状态保存到 `localStorage` 的 `__cnav_theme__`，监控状态保存到 `__cnav_monitor__`。
- 主题切换：支持深色主题和浅色主题切换。
- 拖拽移动：可以拖拽面板标题栏移动面板位置。
- 收起面板：可以将完整面板收起为页面右侧的圆形按钮，并再次点击展开。
- 定位兼容处理：当固定定位异常时，脚本会切换为绝对定位并随滚动更新位置。
- Gemini 兼容扫描：支持常见标签、类名、数据属性、Shadow DOM 遍历和通用容器扫描。
- ChatGPT 兼容扫描：支持 `data-message-author-role` 消息节点和 `conversation-turn` 文章节点。

## 安装方法

1. 在浏览器中安装 Tampermonkey 扩展。
2. 打开 Tampermonkey 管理面板。
3. 新建用户脚本。
4. 将 `chat-navigator.user.js` 的完整内容粘贴到编辑器中。
5. 保存脚本。
6. 打开支持的网站后，脚本会在页面右侧注入对话导航面板。

## 使用说明

- 监控按钮：位于面板标题栏。显示 `⏸` 时表示监控已关闭，点击后启用监控；显示 `▶` 时表示监控已启用，点击后暂停监控。
- 主题按钮：切换深色主题和浅色主题，状态会持久保存。
- 刷新按钮：在监控开启时重新扫描当前页面对话内容；监控关闭时不会扫描页面，只会保持暂停提示。
- 收起按钮：隐藏完整面板，并显示圆形展开按钮。
- 圆形展开按钮：重新打开已收起的导航面板。
- 搜索框：输入关键词后，列表只显示文本中包含该关键词的对话条目。
- 对话列表：每条记录显示序号、角色和内容预览；点击条目会滚动到对应对话位置并短暂高亮。
- 拖拽移动：按住面板标题栏拖动，可以调整面板在页面中的位置。

## 支持的网站

- `https://gemini.google.com/*`
- `https://chatgpt.com/*`
- `https://chat.openai.com/*`
